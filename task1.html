Информация о свойствах CSS

Свойство position определяет тип позиционирования, используемого для элемента из пяти разных значений: 1. static (статический) — означает, что расположение элементов соответствует их расположению в HTML-коде;
2. relative (относительный) — позиционируется относительно своего исходного положения;
3. fixed (фиксированный) — позиция элемента блокируется в определенной позиции на экране;
4. absolute (абсолютный) — позволяет поместить элемент в любой позиции страницы с точностью до одного пиксела;
5. sticky (закреплённый) — позиционируется на основе позиции прокрутки пользователя.
